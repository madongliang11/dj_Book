## 书籍电商平台说明文档


### 主要功能点：

（1）首页

（2）分类页

（3）详情页面

（4）搜索功能（全文检索haystack+whoosh+jieba）

（5）评论功能

（6）用户登录注册

 (7) 购物车功能

（8）订单功能

（9）在线支付（支付宝）

（10）统计页面（采用highcharts进行在线数据统计）

（11）网络爬虫（见tools/scrapy_book）

 (12) celery异步处理功能

 (13) xadmin管理后台（支持富文本操作）

 (14) django-rest-framework提供dashboard和api接口平台
